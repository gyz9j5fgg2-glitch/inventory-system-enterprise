## 服务器上架与初始化

### 上架前检查清单

```
□ 服务器外观检查（无损坏、变形）
□ 配件完整性检查（导轨、电源线、文档）
□ CPU、内存、硬盘安装检查
□ 序列号记录与登记
□ 固件版本确认
□ 机房温度: 18-27°C，湿度: 40-60%
□ 电力供应稳定，UPS就绪
□ 网络布线到位
□ 消防、监控系统正常
```

### 服务器上架步骤

```bash
# 步骤1: 安装导轨
1. 将导轨内轨安装到服务器两侧
2. 将导轨外轨安装到机柜立柱
3. 调整导轨深度匹配机柜
4. 测试抽拉顺畅

# 步骤2: 服务器上架
1. 两人配合将服务器抬入导轨
2. 推入机柜直至卡扣锁定
3. 安装固定螺丝（前后各2颗）
4. 安装理线架

# 步骤3: 连接线缆
1. 电源线连接到PDU（双路）
2. 网线按规划连接到交换机
   - 管理网口 → 管理交换机
   - 业务网口 → 业务交换机
   - 存储网口 → 存储交换机
3. 光纤连接到存储交换机
4. 整理线缆，绑扎固定

# 步骤4: 上电测试
1. 先开PDU电源
2. 再开服务器电源
3. 观察启动状态指示灯
4. 进入BIOS检查硬件识别
```

### BIOS配置要点

```
System Settings → System BIOS
├── Processor Settings
│   ├── Virtualization Technology: Enabled
│   ├── VT for Direct I/O: Enabled
│   └── Sub-NUMA Cluster: Enabled
├── SATA Settings
│   └── Embedded SATA: AHCI Mode
└── Boot Settings
    └── Boot Mode: UEFI
```

### RAID配置

```bash
# 系统盘 RAID1 (2块480GB SSD)
RAID Level: RAID-1
Virtual Drive Name: OS
Strip Size: 256KB
Read Policy: Read Ahead
Write Policy: Write Back

# 缓存盘和数据盘设置为 JBOD/Non-RAID
# 由分布式存储管理
```

## 超融合平台搭建

### 方案A: VMware vSAN (推荐)

#### ESXi安装

```bash
# 1. 制作ESXi 8.0启动U盘
# 2. 从U盘启动安装
# 3. 选择RAID1系统盘作为安装目标
# 4. 设置root密码
# 5. 重启完成安装

# 安装后配置
按F2进入系统配置
├── Configure Management Network
│   ├── IP Address: 192.168.10.11
│   ├── Subnet Mask: 255.255.255.0
│   ├── Default Gateway: 192.168.10.1
│   └── VLAN: 10
├── Troubleshooting Options
│   ├── Enable SSH
│   └── Enable ESXi Shell
└── System → Time & Date
    └── NTP Servers: ntp.aliyun.com
```

#### vCenter与vSAN配置

```bash
# 1. 部署vCenter Server Appliance
# 2. 添加3台ESXi主机到vCenter
# 3. 创建集群，启用vSAN
# 4. 声明磁盘:
#    - 缓存层: 1.92TB NVMe SSD
#    - 容量层: 8TB HDD x 6
# 5. 配置vSAN存储策略
```

### 方案B: Proxmox + Ceph (开源)

```bash
# 1. 安装Proxmox VE 8.0
# 2. 创建集群
pvecm create eims-cluster

# 3. 其他节点加入
pvecm add 192.168.10.11

# 4. 部署Ceph
pveceph install
pveceph init --network 192.168.30.0/24
pveceph mon create

# 5. 创建OSD
ceph orch daemon add osd node1:/dev/sdb
# ... 为所有数据盘创建OSD

# 6. 创建存储池
ceph osd pool create vm-data 128 128 replicated
```

## 应用系统部署

### 数据库服务器 (EIMS-DB)

```bash
# 安装PostgreSQL 15
sudo dnf install -y postgresql15-server
sudo /usr/pgsql-15/bin/postgresql-15-setup initdb

# 配置
listen_addresses = '*'
max_connections = 500
shared_buffers = 8GB

# 启动
sudo systemctl enable postgresql-15
sudo systemctl start postgresql-15

# 创建数据库
CREATE DATABASE inventory;
CREATE USER inventory WITH PASSWORD 'password';

# 安装Redis
sudo dnf install -y redis
sudo systemctl enable redis
sudo systemctl start redis
```

### Web服务器 (EIMS-Web01/02)

```bash
# 安装Docker
sudo dnf install -y docker-ce
sudo systemctl enable docker
sudo systemctl start docker

# 部署应用
cd /opt/inventory-system
git clone https://github.com/gyz9j5fgg2-glitch/inventory-system-enterprise.git .
cp deploy/.env.example deploy/.env
# 编辑.env配置数据库连接

docker compose -f deploy/docker-compose.yml -f deploy/docker-compose.prod.yml up -d
```
